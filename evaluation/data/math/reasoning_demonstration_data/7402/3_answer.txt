Answer: To evaluate \( i^{11} + i^{16} + i^{21} + i^{26} + i^{31} \), we use the fact that the powers of \( i \) cycle every four exponents: \( i, -1, -i, 1 \).

1. \( i^{11} \):
   \[
   11 \div 4 = 2 \text{ remainder } 3 \implies i^{11} = i^3 = -i
   \]

2. \( i^{16} \):
   \[
   16 \div 4 = 4 \text{ remainder } 0 \implies i^{16} = i^0 = 1
   \]

3. \( i^{21} \):
   \[
   21 \div 4 = 5 \text{ remainder } 1 \implies i^{21} = i^1 = i
   \]

4. \( i^{26} \):
   \[
   26 \div 4 = 6 \text{ remainder } 2 \implies i^{26} = i^2 = -1
   \]

5. \( i^{31} \):
   \[
   31 \div 4 = 7 \text{ remainder } 3 \implies i^{31} = i^3 = -i
   \]

summing these results:
\[
i^{11} + i^{16} + i^{21} + i^{26} + i^{31} = -i + 1 + i - 1 - i
\]

combining like terms:
\[
(-i + i - i) + (1 - 1) = -i + 0 = -i
\]

thus, the final answer is:
\[

Answer: To find the value of \((10x - 3)^2\) given the equation \(5x^2 + 4 = 3x + 9\), we start by solving for \(x\).

first, rearrange the equation:
\[5x^2 + 4 = 3x + 9\]
\[5x^2 - 3x - 5 = 0\]

using the quadratic formula \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\) with \(a = 5\), \(b = -3\), and \(c = -5\):
\[x = \frac{3 \pm \sqrt{9 + 100}}{10}\]
\[x = \frac{3 \pm \sqrt{109}}{10}\]

next, we need to find \((10x - 3)^2\). substituting \(x = \frac{3 \pm \sqrt{109}}{10}\):
\[(10 \cdot \frac{3 \pm \sqrt{109}}{10} - 3)^2\]
\[(3 \pm \sqrt{109} - 3)^2\]
\[(\pm \sqrt{109})^2\]
\[109\]

alternatively, we can manipulate the original equation to express \((10x - 3)^2\) directly:
\[5x^2 - 3x = 5\]
\[(10x - 3)^2 = 100x^2 - 60x + 9\]
\[100x^2 - 60x + 9 = 20(5x^2 - 3x) + 9\]
\[20(5) + 9 = 100 + 9 = 109\]

thus, the value of \((10x - 3)^2\) is:
\[\boxed{109}\]
